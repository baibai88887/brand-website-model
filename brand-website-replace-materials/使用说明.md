# 品牌官网替换资料使用说明

## 一、文件夹结构说明

```
brand-website-replace-materials/
├── images/           # 存放所有需要替换的图片文件
├── text/             # 存放所有需要替换的文本内容
├── config/           # 存放配置文件
└── replace-list.csv  # 替换内容清单
```

## 二、如何替换内容

### 1. 图片资源

**需要准备的图片文件：**

| 文件名 | 用途 | 格式要求 | 尺寸建议 |
|-------|------|---------|---------|
| logo.png | 品牌Logo | PNG/SVG | 500x500像素 |
| hero-bg.jpg | 首页Hero背景图 | JPG | 1920x1080像素 |
| product-1.jpg | 产品1主图 | JPG/PNG | 800x800像素 |
| product-2.jpg | 产品2主图 | JPG/PNG | 800x800像素 |
| product-3.jpg | 产品3主图 | JPG/PNG | 800x800像素 |
| product-4.jpg | 产品4主图 | JPG/PNG | 800x800像素 |
| news-1.jpg | 新闻1配图 | JPG | 800x400像素 |
| news-2.jpg | 新闻2配图 | JPG | 800x400像素 |
| news-3.jpg | 新闻3配图 | JPG | 800x400像素 |
| team-1.jpg | 团队成员1照片 | JPG | 300x300像素 |
| team-2.jpg | 团队成员2照片 | JPG | 300x300像素 |
| favicon.ico | 浏览器图标 | ICO | 16x16, 32x32像素 |
| icon-192.png | PWA图标 | PNG | 192x192像素 |
| icon-512.png | PWA图标 | PNG | 512x512像素 |

**替换方法：**
1. 将准备好的图片文件放入 `images/` 文件夹
2. 确保文件名与上述表格中的文件名一致
3. 图片文件大小建议控制在200KB以内

### 2. 文本内容

**已提供的文本模板：**

| 文件名 | 用途 | 格式 | 替换方法 |
|-------|------|------|---------|
| products.csv | 产品列表 | CSV | 替换表格中的示例数据 |
| news.csv | 新闻列表 | CSV | 替换表格中的示例新闻 |
| history.csv | 发展历程 | CSV | 替换表格中的示例时间线 |
| team.csv | 管理团队 | CSV | 替换表格中的示例团队成员 |
| faqs.csv | 常见问题 | CSV | 替换表格中的示例问题 |
| social.csv | 社交媒体链接 | CSV | 替换表格中的示例链接 |
| about-intro.txt | 公司简介 | TXT | 替换文件中的示例内容 |

**CSV文件编辑说明：**
- 使用Excel或WPS表格打开CSV文件
- 不要修改第一行的标题
- 每个字段内容不要包含逗号（,）
- 保存时选择CSV格式

**TXT文件编辑说明：**
- 使用记事本或其他文本编辑器打开
- 直接替换文件内容
- 保持格式清晰，段落分明

### 3. 配置文件

**需要修改的配置项：**

| 配置项 | 替换位置 | 替换方法 |
|-------|---------|---------|
| 品牌名称 | `app/layout.tsx` | 将"BRAND"替换为您的品牌名称 |
| Slogan | `app/layout.tsx` | 将"极致创新，激发可能"替换为您的Slogan |
| 网站域名 | `app/layout.tsx` | 将"https://www.brand.com"替换为您的真实域名 |
| 网站域名 | `public/robots.txt` | 将"https://www.brand.com"替换为您的真实域名 |
| 网站域名 | `app/sitemap.ts` | 将"https://www.brand.com"替换为您的真实域名 |
| 主题颜色 | `tailwind.config.js` | 将"primary: #0ea5e9"替换为您的品牌色 |
| SEO关键词 | `app/layout.tsx` | 将关键词列表替换为您的品牌关键词 |

## 三、替换优先级

1. **最高优先级**：品牌Logo、网站域名、联系信息
2. **次高优先级**：首页Hero内容、产品列表、公司简介
3. **中等优先级**：新闻内容、团队信息、核心价值观
4. **低优先级**：发展历程、常见问题、社交媒体链接

## 四、注意事项

1. **图片质量**：所有图片必须清晰、高质量，避免模糊或变形
2. **文本准确性**：确保所有文本内容准确无误，符合公司实际情况
3. **格式一致性**：保持CSV文件格式一致，不要修改列数和标题
4. **版权问题**：确保所有图片和内容都有合法版权，避免侵权
5. **及时更新**：网站上线后，定期更新新闻、产品等内容

## 五、交付方式

完成所有替换后，请将整个`brand-website-replace-materials`文件夹压缩为ZIP文件，通过指定方式交付。
